# Blackjack Game

A simple command-line implementation of the classic Blackjack card game in Python.

## Features

- Play against the computer
- Standard Blackjack rules with Ace handling (11 or 1)
- ASCII art logo
- Interactive gameplay

## How to Play

1. Run the `main.py` file
2. Choose to play or not
3. Get dealt cards and decide to hit or stand
4. Computer plays automatically
5. See the results

## Requirements

- Python 3.x

## Running the Game

```bash
python main.py
```

## Game Rules

- Goal: Get as close to 21 as possible without going over
- Face cards are worth 10, Ace is 11 or 1
- Blackjack: 21 with exactly 2 cards
- Dealer hits on 16, stands on 17

## Files

- `main.py`: Main game logic
- `art.py`: ASCII art for the logo
