# Rock Paper Scissors Game

A simple Rock Paper Scissors game implemented in Python.

## Description

This is a classic Rock Paper Scissors game where you play against the computer. Choose rock, paper, or scissors, and see if you can beat the computer!

The game features ASCII art representations of the choices.

## How to Play

1. Run the `task.py` script.
2. Enter 0 for rock, 1 for paper, or 2 for scissors.
3. The computer will randomly choose.
4. See the ASCII art and the result.

## Rules

- Rock beats Scissors
- Scissors beats Paper
- Paper beats Rock
- Same choice = Draw

## Requirements

- Python 3.x

## Running the Game

```bash
python task.py
```

## GUI Version

A simple GUI version is available in `gui.py` using Tkinter.

Run with:

```bash
python gui.py
```
